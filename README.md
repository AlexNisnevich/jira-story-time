jira-story-time
===============

Notes: 
- in `src/story.js` if you set `Story.devMode = false;` it will start updating story points.
- in `src/story.js` if you set `Story.autoUpdate = flase;` it will stop auto updating from server.

Development
===
Dependencies
- coffeescript
  - `npm install -g coffee-script`
  
Run reloader script
```
  ./reload_extension.rb
```
